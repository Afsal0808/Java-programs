# 📅 Day 02 – Array Problems (Company Level)

## 🔥 Problems Covered

### 1️⃣ Move Negative Elements to End

* Rearrange array so that all negative elements move to the end.
* Used **Two Pointer Technique**
* Time Complexity: `O(n)`
* Space Complexity: `O(1)`

---

### 2️⃣ Majority Element (> n/2 times)

* Find element appearing more than `n/2` times.
* Used **Boyer-Moore Voting Algorithm**
* Time Complexity: `O(n)`
* Space Complexity: `O(1)`

---

### 3️⃣ Smallest Missing Positive Number

* Find smallest missing positive integer.
* Used **Index Marking Technique**
* Time Complexity: `O(n)`
* Space Complexity: `O(1)`
